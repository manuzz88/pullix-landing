# pullix.fit

Landing page di **pullix X1** — la prima palestra portatile ricaricabile al mondo.

Sito statico servito via GitHub Pages su [pullix.fit](https://pullix.fit).

## Struttura

- `index.html` — landing page italiana
- `en/index.html` — landing page inglese
- `styles.css` / `site.js` — stile e script condivisi tra le due lingue
- `assets/` — foto prodotto e video
- `og-image.jpg` — anteprima social (Open Graph)
- `favicon.svg` — favicon
- `CNAME` — dominio custom per GitHub Pages
- `robots.txt` / `sitemap.xml` — SEO

## Deploy

Ogni push su `main` va live automaticamente su pullix.fit.
